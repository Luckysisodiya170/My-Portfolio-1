# Lucky Sisodiya | Full Stack Developer Portfolio

This is my professional portfolio website built using **React** and **Tailwind CSS**. It showcases my projects, skills, and experience as a Full Stack Developer.

## 🔗 Live Link
**Check it out here:** [https://my-portfolio-1-livid.vercel.app](https://my-portfolio-1-livid.vercel.app)

## ✨ Key Features
- **Modern UI:** Minimalist dark theme with smooth Framer Motion animations.
- **2x2 Skills Grid:** Showcasing Frontend, Backend, Database, and DevOps.
- **Projects Showcase:** Real-world projects like TRICKSY and MediAdmin.
- **Fully Responsive:** Pixel-perfect on all screen sizes.

## 🛠️ Tech Stack
- **Frontend:** React.js, Tailwind CSS, Framer Motion
- **Tools:** Git, GitHub, Vercel, Railway, Postman
- **Database Expertise:** MySQL, PostgreSQL, MongoDB
